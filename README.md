# FRC-Event-Research-System
提供團隊在FRC比賽時調查各隊資料
使用Google Firebase

#### 如何加入
1. 先下載專案以及相關需求軟體
要Deploy到firebase需要安裝 Node.js & npm
2. 於專案的資料夾安裝Firebase CLI  
輸入npm install -g firebase-tools安裝
3. 切入functions資料夾安裝所需插件  
輸入    npm install 安裝所需插件
4. 登入Firebase CLI  
輸入    firebase login 登入
